# Microbiome Lab @ CBNU Website

This is the official website for the Microbiome Lab at Chungbuk National University. It provides information about the lab, its research, members, publications, and news.

## Features

*   **Responsive Design:** The website is designed to work on all devices, from desktops to mobile phones.
*   **Dynamic Content:** The website uses JavaScript to load some content dynamically, such as the navigation bar, blog posts, and visitor information.
*   **Interactive Elements:** The website includes interactive elements such as modals for viewing course syllabi.
*   **Visitor Counter:** The website includes a visitor counter to track the number of visitors.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

You will need a web browser to view the website.

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/mblcbnu/mblcbnu.github.io.git
    ```
2.  Open the `index.html` file in your web browser.

## Folder Structure

*   `assets/`: Contains the CSS and JavaScript files for the website.
*   `blogs/`: Contains the HTML files for the blog posts.
*   `projects/`: Contains the HTML files for the research projects.
*   `*.html`: The HTML files for the main pages of the website.
*   `*.png`, `*.jpg`: Image files.
*   `news_data.json`: A JSON file containing the data for the blog posts.
